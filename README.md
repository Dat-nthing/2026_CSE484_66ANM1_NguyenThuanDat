## ✨ Giới thiệu

Repo này lưu trữ các bài thực hành về truy vấn dữ liệu, thiết kế lược đồ quan hệ và các bài tập chuẩn hóa dữ liệu. Đây là nền tảng để mình hiểu cách lưu trữ, quản lý và truy xuất thông tin một cách tối ưu và bảo mật.

✅ Thiết kế ERD — ✅ Truy vấn SQL nâng cao — ✅ Chuẩn hóa dữ liệu (1NF-3NF/BCNF) — ✅ Quản trị Database

---

## 🎯 Mục tiêu
- Thành thạo ngôn ngữ truy vấn cấu trúc (**SQL**) để thao tác với dữ liệu (CRUD).
- Hiểu và áp dụng mô hình dữ liệu quan hệ (Relational Model).
- Kỹ năng phân tích và thiết kế sơ đồ thực thể mối quan hệ (**ERD**).
- Nắm vững các quy tắc chuẩn hóa để tránh dư thừa và mâu thuẫn dữ liệu.
- Hiểu về Transaction, Index và tối ưu hóa hiệu năng truy vấn.

## 🧰 Công cụ & Hệ quản trị CSDL

- **DBMS**: MySQL, Microsoft SQL Server hoặc PostgreSQL.
- **Thiết kế sơ đồ**: MySQL Workbench, Draw.io (để vẽ ERD).
- **Công cụ hỗ trợ**: DBeaver, TablePlus (để quản lý Database trực quan).
- **Ngôn ngữ bổ trợ**: Python/Node.js (kết nối và thao tác với Database qua code).

---

## 🚀 Nội dung thực hành

1.  **Thiết kế:** Xây dựng mô hình ERD từ yêu cầu thực tế và chuyển đổi sang lược đồ quan hệ.
2.  **Truy vấn:** Thực hiện các câu lệnh `SELECT`, `JOIN`, `GROUP BY`, `HAVING` phức tạp.
3.  **Lập trình trong CSDL:** Viết View, Stored Procedure, Function và Trigger.
4.  **Tối ưu hóa:** Sử dụng Index để tăng tốc độ truy vấn và xử lý tranh chấp dữ liệu (Locking).
